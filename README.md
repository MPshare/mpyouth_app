 # 마포유스 앱

![image](https://imgur.com/UKsb1Gk.gif)  
- 마포유스 앱은 마포구 내 4개 청소년 기관의 요청으로 개발된 **안드로이드 네이티브 어플리케이션**입니다.
- 마포구 청소년들이 기관에서 운영되는 `봉사활동` `평생교육` `재능기부` `청소년 프로그램`에 관한 정보를 한 곳에서 쉽게 확인하고 신청할 수 있습니다.
